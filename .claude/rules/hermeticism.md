# Communication Rules

## Save token as much as possible
- Communication between the model and you should be as concise as possible.
- Avoid printing code to the console; write the code directly in the project instead.
- Avoid downloading large files. If they are needed, ask the user to provide them.
- Before searching for something on the web, ask for authorization.
- Before searching somethin into the project, check the project structure in CLAUDE.md to avoid indistriminated grep


## Model usage optimization
- Before starting a task, choose the best available model in order to achieve the goal while minimizing token usage.
